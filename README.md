## 🎭 The Aesthetic: Gojo vs Sukuna
Limitless Store isn't just a shop; it's a domain expansion. Built with a dual-themed cinematic interface, users choose their side upon entry:
- **🔵 The Honored One (Gojo)**: A clean, high-contrast light/cyan aesthetic representing the Infinite Void.
- **🔴 King of Curses (Sukuna)**: A dark, visceral red/black aesthetic representing the Malevolent Shrine.

Every interaction, from the custom cursed energy cursors to the parallax scroll reveals, is designed to provide an elite user experience.

---

## ✨ Key Features

- **🛡️ Domain-Level Security**: Secure checkout flow integrated with Google Sheets for robust order tracking.
- **📈 Automated Order Management**: Orders are synchronized in real-time to a professional Google Sheet via **Google Apps Script**.
- **📩 Cursed Resonance Contact**: A professional contact form powered by **Formspree** for reliable customer inquiries.
- **🎨 Immersive UI**: HTML5 Canvas particle systems, custom Framer Motion animations, and dynamic domain-switching mechanics.
- **📱 Universal Technique**: Fully responsive architecture optimized for desktop, tablet, and mobile devices.

## 🚀 Cursed Technique Setup

To manifest this project locally, follow these steps:

### 1. Clone the Repository
```bash
git clone https://github.com/ahmed-hmida/Limitless-Store.git
cd Limitless-Store
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Environment Configuration
Create a `.env.local` file in the root and add your Supabase and Formspree credentials:
```env
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
```

### 4. Manifest the Dev Server
```bash
npm run dev
```
Navigate to `http://localhost:3000` to see the domain expansion in action.
